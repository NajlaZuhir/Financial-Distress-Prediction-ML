# Financial-Distress-Predictor

## Problem Definition

**Problem Type:** Classification  

**Objective:**  
Predict whether a company is financially distressed, i.e., struggling to pay employees, bills, or meet other financial obligations.  

**Input Features:**  
Top financial and operational factors that significantly impact a company's financial health.  

**Output:**  
Binary classification:  
- `1` → Financially Distressed  
- `0` → Not Financially Distressed  

**Evaluation Metrics:**  
Standard classification metrics such as:  
- Accuracy  
- Precision  
- Recall  
- F1-score  
- ROC-AUC (optional for deeper analysis)
